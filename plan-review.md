# 니가가라투자왕 문서 검토 메모

## 확인한 오류와 보완점

1. 파일 인코딩 확인 필요
   - 기존 기획 문서를 PowerShell 기본 설정으로 열면 한글이 깨져 보일 수 있다.
   - 최종 문서는 UTF-8로 저장하고, 협업 문서로 옮길 때도 UTF-8 인코딩을 유지하는 것이 좋다.

2. 리그 수익률 공식 괄호 보완
   - 잘못 읽히기 쉬운 식: `(총 평가금액 - 초기 가상 투자금) / 초기 가상 투자금 * 100`
   - 앱 구현에서는 아래 공식으로 고정한다.

```text
리그 수익률 = ((총 평가금액 - 초기 가상 투자금) / 초기 가상 투자금) * 100
```

3. 랭킹 기준 확정 필요
   - 친구 리그에서는 개별 종목 수익률보다 전체 자산 기준이 더 적합하다.
   - 앱 구현 기준:

```text
총 자산 = 보유 현금 + 보유 종목 평가금액
랭킹 = 총 자산 높은 순
동률 = 수익률 높은 순, 이후 닉네임 가나다순
```

4. 개인정보 없는 참여 방식의 한계
   - 이메일, 전화번호, 실명 없이 참여하면 진짜 사용자 식별이 약하다.
   - MVP에서는 브라우저 localStorage 기반으로 저장한다.
   - 실제 서비스에서는 초대 코드와 기기별 임시 토큰을 함께 저장하는 방식이 필요하다.

5. 가격 데이터 정책 필요
   - 실제 실시간 가격은 유료 API 또는 사용 제한이 많다.
   - 이번 프로토타입은 샘플 종목 가격을 사용한다.
   - 실제 배포 전에는 지연 시세, 종가 데이터, 또는 정식 시세 API를 선택해야 한다.

6. 리그 종료 처리 보완
   - 종료 후 매수/매도가 막혀야 한다.
   - 종료 시점의 가격 기준으로 최종 랭킹을 고정해야 한다.
   - 이번 앱에서는 "리그 종료" 버튼을 누르면 즉시 종료되고 거래가 잠긴다.

## 이번 프로토타입 범위

- 모바일 우선 웹앱
- 제목: 니가가라투자왕
- 리그 생성
- 초대 코드 표시
- 닉네임으로 참여
- 참여자에게 10,000,000원 지급
- 샘플 주식으로 가상 매수/매도
- 현금, 평가금액, 수익률 계산
- 리그 랭킹
- 리그 종료 및 거래 잠금
